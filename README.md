## 📌 Project Overview

Teen Enslavement is a modern awareness website designed to educate teenagers about different forms of addiction — including drugs, social media, and gaming — and guide them toward recovery resources. Built by students of SRKR Engineering College, the platform focuses on spreading awareness, encouraging early intervention, and supporting mental health.

---

## ✨ Features

* 🎯 Clean and modern UI/UX
* 📱 Fully responsive design
* 📚 Informative sections on drug, social media, and gaming addiction
* 🎥 Motivational video section
* 📍 De-addiction centres with Google Maps integration
* 📞 Emergency support via Tele-MANAS helpline
* 🧭 Dynamic navigation bar
* ✨ Scroll-based animations
